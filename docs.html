Getting Started with TigerFlow

Welcome to TigerFlow! This guide will help you get up and running with the most powerful SEO analyzer for Webflow Designer. Follow these simple steps to start optimizing your pages for search engines.

Installation

1

Open Webflow Designer

Launch Webflow Designer and open any project you want to optimize.

2

Access Extensions

Click on the Extensions icon in the left sidebar or press E on your keyboard.

3

Find TigerFlow

Search for "TigerFlow" in the extensions marketplace or browse the SEO category.

4

Install & Launch

Click "Install" and then "Open" to launch TigerFlow. The extension will appear as a floating panel.

First Time Setup

1

Set Your Focus Keyword

Enter your target keyword for the page. This is the main term you want to rank for in search results. TigerFlow will save this automatically for future sessions.

2

Review Your SEO Score

TigerFlow instantly analyzes your page and provides a score from 0-100. Green (70+) is good, yellow (40-69) needs work, and red (below 40) requires immediate attention.

3

Follow Recommendations

Click on the "SEO Recommendations" section to see specific actions you can take to improve your score.

Start with the "To Fix" tab in recommendations to focus on the most critical issues first. Once those are resolved, check the "Optimized" tab to see what you've already done well!

Frequently Asked Questions

What is an SEO score and how is it calculated?
The SEO score is a comprehensive metric from 0-100 that evaluates your page's search engine optimization. It's calculated based on:
  • Meta tags (title, description) - 25%
  • Content quality and length - 25%
  • Heading structure - 25%
  • Images and alt text - 25%
Each component is weighted and combined to give you an overall score. A score of 70+ is considered good for SEO.
Why is my focus keyword important?
Your focus keyword is the main search term you want your page to rank for. TigerFlow checks if this keyword appears in crucial SEO locations:
  • Page title (most important)
  • H1 heading
  • First paragraph of content
  • URL slug
  • Meta description
  • Image alt text
Strategic keyword placement helps search engines understand what your page is about.
How often should I run TigerFlow analysis?
Run TigerFlow analysis:
  • Every time you create a new page
  • After making significant content changes
  • When updating meta tags or headings
  • Before publishing any page
  • During regular SEO audits (monthly recommended)
The refresh button (‚Üª) lets you re-analyze after making changes without closing the extension.
What's the ideal content length for SEO?
Content length recommendations vary by page type:
  • Landing pages: 500-800 words
  • Blog posts: 1,500+ words
  • Product pages: 300-500 words
  • Pillar content: 2,000+ words
TigerFlow recommends a minimum of 300 words for any page, with 600+ being ideal for most content.
Can TigerFlow help with technical SEO?
Yes! TigerFlow analyzes several technical SEO factors:
  • URL structure and slug optimization
  • Image optimization (alt text, file names)
  • Internal and external link analysis
  • Heading hierarchy (H1-H6 structure)
  • Meta tags and Open Graph data
  • Content readability scores
For advanced technical SEO like schema markup and canonical tags, upgrade to our Pro or Agency plans.
Why does TigerFlow load faster than other SEO tools?
TigerFlow uses an optimized single-pass DOM traversal algorithm that collects all page data in one scan. This means:
  • Near-instant loading (under 1 second for most pages)
  • Minimal impact on Designer performance
  • Efficient caching of analyzed data
  • Smart refresh that only updates changed content
We've optimized every aspect for the best performance in Webflow Designer.

How-To Guides

How to Improve Your SEO Score

1

Fix Critical Errors First

Go to SEO Recommendations ‚Üí "To Fix" tab. Address red error items before warnings. Common quick wins:

  • Add missing title tags
  • Write meta descriptions
  • Set H1 headings
  • Add alt text to images
2

Optimize Content Length

Check the Content Analysis section. If word count is low:

  • Expand existing sections with more detail
  • Add FAQ sections
  • Include relevant examples or case studies
  • Create comprehensive guides
3

Perfect Your Keyword Strategy

Ensure your focus keyword appears in:

  • Title tag (near the beginning)
  • H1 heading (exactly once)
  • First 100 words of content
  • At least one H2 heading
  • 2-3 times naturally in body content

How to Optimize Images for SEO

1

Add Descriptive Alt Text

Every image needs alt text that:

  • Describes what's in the image
  • Includes your keyword (where relevant)
  • Is 125 characters or less
  • Doesn't start with "image of" or "picture of"
2

Use SEO-Friendly File Names

Before uploading images:

  • Rename files descriptively (not IMG_1234.jpg)
  • Use hyphens between words
  • Include keywords where appropriate
  • Keep names short and relevant

Example: webflow-seo-best-practices.jpg

How to Structure Content for SEO

1

Create Logical Heading Hierarchy

Proper heading structure:

  • One H1 per page (your main title)
  • H2s for major sections
  • H3s for subsections under H2s
  • Never skip levels (H1 ‚Üí H3)
2

Optimize Paragraph Length

For better readability:

  • Keep paragraphs to 3-4 sentences
  • Use bullet points for lists
  • Add white space between sections
  • Break up long content with subheadings

How to Build Quality Links

1

Internal Linking Strategy

Connect related pages:

  • Link to relevant service/product pages
  • Use descriptive anchor text (not "click here")
  • Create topic clusters with pillar content
  • Add 3-5 internal links per page
2

External Link Best Practices

When linking to other sites:

  • Only link to reputable, relevant sources
  • Use 2-4 external links for most content
  • Open external links in new tabs
  • Consider adding nofollow for affiliate links

  • Don't duplicate title tags across pages
  • Don't use generic meta descriptions
  • Don't ignore mobile optimization
  • Don't forget to set up 301 redirects when changing URLs

SEO Glossary

Alt Text (Alternative Text)
Descriptive text that explains what an image contains. Used by screen readers for accessibility and by search engines to understand image content. Critical for image SEO.
Anchor Text
The clickable text in a hyperlink. Good anchor text is descriptive and includes relevant keywords, helping both users and search engines understand what the linked page is about.
Canonical Tag
An HTML element that tells search engines which version of a page is the "main" version when duplicate content exists. Helps prevent duplicate content penalties.
Content Quality Score
A metric that evaluates how well-written and comprehensive your content is. Factors include word count, readability, sentence structure, and topic coverage.
E-E-A-T
Experience, Expertise, Authoritativeness, and Trustworthiness. Google's framework for evaluating content quality, especially important for YMYL (Your Money or Your Life) topics.
Focus Keyword
The primary search term you want a specific page to rank for. Should appear naturally in key locations like the title, headings, and throughout the content.
H1 Tag
The main heading on a page, typically the page title. Each page should have exactly one H1 tag that includes your focus keyword when relevant.
Heading Hierarchy
The structured organization of headings (H1-H6) on a page. Proper hierarchy helps search engines understand content structure and improves user experience.
Internal Links
Links from one page on your website to another page on the same website. Help distribute page authority and guide users through related content.
Meta Description
A brief summary of a page's content that appears in search results. Should be 120-160 characters and include your focus keyword to improve click-through rates.
Nofollow Attribute
An HTML attribute (rel="nofollow") that tells search engines not to pass link authority to the linked page. Often used for sponsored or user-generated content links.
Open Graph Tags
Meta tags that control how your content appears when shared on social media platforms. Include og:title, og:description, og:image, and more.
Readability Score
A measure of how easy your content is to read and understand. TigerFlow uses the Flesch Reading Ease formula, aiming for scores between 30-70 for most web content.
Reading Time
Estimated minutes needed to read the content on a page. Calculated based on average reading speed (200-250 words per minute). Helps users gauge content depth.
Schema Markup
Structured data vocabulary that helps search engines understand your content better. Can result in rich snippets in search results (ratings, prices, events, etc.).
SEO Score
TigerFlow's proprietary 0-100 metric that evaluates overall page optimization. Combines meta tags, content quality, technical factors, and keyword usage into one score.
Title Tag
The HTML element that specifies the title of a web page. Appears in browser tabs and search results. Should be 30-60 characters and include your focus keyword.
URL Slug
The part of a URL that comes after the domain name. Should be short, descriptive, include keywords, and use hyphens to separate words (e.g., /seo-best-practices).
Word Count
Total number of words on a page. While quality matters more than quantity, most pages should have at least 300 words, with 600+ being ideal for ranking.

Troubleshooting Common Issues

TigerFlow is loading slowly
  • Check your internet connection
  • Try refreshing the page in Designer
  • Clear your browser cache
  • Make sure you're using the latest version of TigerFlow
  • For very large pages (100+ elements), loading may take 2-3 seconds
My SEO score seems incorrect
  • Click the refresh button (‚Üª) to re-analyze
  • Make sure all your changes are saved in Designer
  • Check that your focus keyword is set correctly
  • Review each section of the analysis for specific issues
  • Remember that scores update in real-time as you make changes
Focus keyword isn't saving
  • Make sure you press Enter after typing your keyword
  • Check that your browser allows localStorage
  • Try closing and reopening the extension
  • Keywords are saved per page - each page has its own keyword
Can't see all recommendations
  • Use the tabs (All/To Fix/Optimized) to filter recommendations
  • Scroll within the recommendations section
  • Completed optimizations move to the "Optimized" tab
  • Click on any recommendation for more details
Extension window is too small
  • TigerFlow uses Webflow's "large" window size for maximum space
  • Unfortunately, Webflow extensions cannot be manually resized
  • Use the collapsible sections to focus on what you need
  • The compact design ensures all features fit comfortably
